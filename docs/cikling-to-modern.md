# 戚林八音 转换成 现代福州市区音系

下表中：
* **舒声紧韵** 指的是 上平、上上、下平。现代福州市区的 55, 33, 53。
* **入声紧韵** 指的是 下入。现代福州市区的 h5/k5。
* **舒声松韵** 指的是 上去、下去。现代福州市区的 213, 242。
* **入声松韵** 指的是 上入。现代福州市区的 h24/k24。
